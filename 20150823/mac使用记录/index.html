支持git命令补全

mac下本来终端命令是支持按tab键自动补全的，但是有些第三方工具是不支持的，比如git。

但是可以通过一些额外的工具让他支持，因为我在平时工作中使用git的频率比较高，每次都要把命令敲全了感觉很low，于是乎，按照下面的步骤做，mac终端也能支持git命令补全了。

第一步
  • 首先安装mac神器homebrew（具体搜索下怎么安装，很简单）。
  • 然后使用homebrew安装一个工具：bash-completion
    • brew install bash-completion
  • 然后查看使用方法：brew info bash-completion
  • 复制从if开头fi结尾的一段代码。
第二步
  • 将这个文件copy下来：https://github.com/git/git/blob/master/contrib/completion/git-completion.bash。
  • 在用户跟目录下新建文件.git-completion.bash，将刚才复制的文件写进去。
第三步
  • 修改.bash_profile，将source ~/.git-completion.bash加入进去。
  • 执行source ~/.bash_profile生效。

常用

  • mysql数据库备份 -> mysqldump -u 用户名 -p 数据库名称 > path/name.sql
  • mysql导入数据 -> source 文件目录地址/example.sql
  • 服务器和本地的文件互换 -> scp uaername@domain:path local
    • username：服务器用户名
    • domain：服务器地址（或ip）
    • path：需要copy的文件在服务器上的目录地址
    • local：需要存放到本地的位置
  • Permission define(public key)错误，由于ssh授权失效，执行ssh-agent && ssh-add ~/.ssh/id_rsa重新授权即可解决。
  • 启动mysql服务 -> mysql.server start
